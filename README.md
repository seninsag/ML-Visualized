# ML Visualized

ML Visualized is a personal project where I'm creating animations to explain Machine Learning concepts using Manim.

The idea is simple: instead of learning through pages of equations, understand concepts by seeing them.

## Current Progress

- Linear Decision Boundaries
- Understanding XOR

## Planned Topics

- Feature Transformations
- Perceptron
- Neural Networks
- Backpropagation
- Gradient Descent

## Built With

- Python
- Manim Community Edition

---

This project is a work in progress, and I'll continue adding new concepts and animations over time.
